### Netlify Sign-up Form

## What does this do?

1. It creates a simple html file for Netlify to build from
2. It facilitates the right inline attributes for forms so that Netlify can append on logic for it to work as a real form
3. It means free email sign-up (for up to 100 users per month!)

## What do I need to set up on Netlify?

1. Follow this guide (it's not that short, but its super simple)
   https://docs.netlify.com/forms/setup/
